# Tài liệu học tập: Module, Import và OOP trong Python

## Giới thiệu

Đây là tài liệu học tập về:
1. **Tổ chức mã nguồn & Cơ chế Import** trong Python
2. **Lập trình hướng đối tượng (OOP)** với Python

Tài liệu được thiết kế cho học sinh mới bắt đầu, với các ví dụ thực tế và dễ hiểu.

## Hướng dẫn sử dụng

### PHẦN 1: Module và Import

1. **Đọc lý thuyết**: Bắt đầu với `phan_1_module_import/LY_THUYET.md`
2. **Thực hành**: Làm theo thứ tự các ví dụ:
   - `vi_du_co_ban/` - Hiểu cơ chế import
   - `vi_du_import/` - Các kiểu import
   - `vi_du_name_main/` - Sử dụng `__name__ == "__main__"`
   - `vi_du_package/` - Tổ chức thành package
3. **Tránh lỗi**: Đọc `loi_thuong_gap/` để biết các lỗi thường gặp
4. **Làm bài tập**: Thực hành với `bai_tap_thuc_hanh/`

### PHẦN 2: OOP

1. **Đọc lý thuyết**: Bắt đầu với `phan_2_oop/LY_THUYET.md`
2. **Thực hành**: Làm theo thứ tự các ví dụ:
   - `vi_du_co_ban/` - Class và Object
   - `vi_du_ke_thua/` - Kế thừa
   - `vi_du_dong_goi/` - Đóng gói
3. **Dự án tổng hợp**: Học từ `du_an_tra_sua/` - Dự án Quán Trà Sữa
4. **Mở rộng**: Làm bài tập trong `bai_tap_mo_rong/` và `bai_tap_thuc_hanh/`

## Yêu cầu

- Python 3.6 trở lên
- Không cần thư viện bên ngoài (chỉ dùng thư viện chuẩn)

## Cách chạy ví dụ

Mỗi thư mục ví dụ đều có file `README.md` hướng dẫn chi tiết cách chạy.

Ví dụ:
```bash
cd phan_1_module_import/vi_du_co_ban
python main.py
```

## Mục tiêu học tập

### Sau PHẦN 1, học sinh sẽ:
- ✅ Hiểu bản chất của Module trong Python
- ✅ Biết cách import và sử dụng module
- ✅ Sử dụng `if __name__ == "__main__":` đúng cách
- ✅ Tổ chức code thành package
- ✅ Tránh được các lỗi thường gặp

### Sau PHẦN 2, học sinh sẽ:
- ✅ Hiểu Class, Object, Attribute, Method
- ✅ Sử dụng Kế thừa để tránh code lặp lại
- ✅ Hiểu và áp dụng Đóng gói (Encapsulation)
- ✅ Thiết kế và xây dựng hệ thống OOP hoàn chỉnh
